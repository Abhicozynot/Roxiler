# Roxiler - MERN Stack Coding Challenge

### Project Setup

1. **Clone the repository:**
   ```bash
   git clone <repository_url>
   cd <repository_folder>
   ```

2. **Install the dependencies:**
   ```bash
   npm install
   ```

3. **Run the backend server:**
   ```bash
   cd backend
   nodemon index.js
   ```

4. **Run the frontend server:**
   ```bash
   cd frontend
   npm run dev
   ```
